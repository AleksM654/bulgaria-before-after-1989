# Bulgaria Before and After 1989

## Working title
Are the trends in selected demographic and economic indicators statistically different between the communist period and the post-communist period?

## Project description
This project will examine whether selected demographic and economic indicators in Bulgaria differ between the communist and post-communist periods. The analysis will focus on long-term trends, data cleaning and consolidation from at least two independent sources, exploratory data analysis, and statistical testing.

## Planned indicators
- Population
- Fertility rate or birth rate
- Infant mortality
- Life expectancy
- GDP per capita
